# cBioPortal for Ruby 
[cBioPortal](http://www.cbioportal.org) for Ruby.

## Installation

```bash
git clone https://github.com/kojix2/ruby_cbioportal
cd ruby_cbioportal
bundle install
bundle exec rake install
```
